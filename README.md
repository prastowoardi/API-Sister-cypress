# API-Mudo-Cypress

Url API mudo 
`
https://api.mudoapi.tech
`

Install package  
```bash
npm install
```

How to run project
```bash
npx cypress open
```

How to get report
```bash
npx cypress run
```

How to review report
1. Open the "testReports" folder.
2. Open the file "index.html" with a live server.
3. Review the reports
